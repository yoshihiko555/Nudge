# ADR-0006: グローバルショートカットは実装しない

## 決定

1. 非アクティブ時に効くグローバルショートカット機能は現時点で実装しない
2. cgo/Carbon などのネイティブ API 連携は採用しない
3. ショートカット設定 UI / 設定項目は追加しない

## 理由

- ネイティブ API 連携は許容しない方針のため
- Wails の KeyBindings はアクティブ時のみで要件を満たさない
- 外部ライブラリ導入による依存・保守・将来互換のリスクを避ける

## 代替案

- Carbon の `RegisterEventHotKey` で実装する
- グローバルホットキーの外部ライブラリを導入する
- macOS のショートカットアプリ/サービス連携で代替する
- アクティブ時のみのショートカットを提供する

## 影響

- 設定画面のショートカット項目は見送り
- `config.json` への設定追加はなし
- 依存追加はなし
- 将来方針が変わった場合は本 ADR を更新する
