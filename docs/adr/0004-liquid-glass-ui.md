# ADR-0004: リキッドグラスデザインの採用と中断タブの廃止

## 決定

1. UI デザインを Apple のリキッドグラス風デザインに刷新する
2. 「中断」タブを廃止し、タブを「進行中」「設定」の 2 つに簡素化する

## 理由

### リキッドグラスデザイン

- macOS ネイティブアプリとしての統一感を高めるため
- Apple の最新デザイン言語に合わせ、モダンな外観を実現
- `backdrop-filter: blur()` によるフロストガラス効果で視認性と美しさを両立

### 中断タブの廃止

- 中断されたタスクに対して操作を行うユースケースがない
- UI をシンプルに保ち、本来の目的（進行中タスクの確認）に集中させる
- 設定画面の「中断の値」フィールドはバックエンド連携のため維持

## 代替案

- Material Design 3 の採用 → macOS との親和性を優先し不採用
- 中断タブを残しつつ非表示にする → 複雑性が増すため不採用

## 影響

- `styles.css`: リキッドグラス用の CSS 変数・スタイルを追加
- `index.html`: 中断タブ・ペインを削除、タブを 2 カラムに変更
- `app.js`: 中断ビュー関連のロジックを削除・簡素化
- ダークモード対応を維持
