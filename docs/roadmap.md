# ロードマップ

## MVP スコープ
- メニューバー常駐 / 小窓表示（クリックでトグル）
- Notion トークン設定
- Database 1件のみ対応
- 進行中タスク取得・表示（Status非表示）
- 中断一覧への一時切替（メニュー）と「戻す」操作
- 完了 / 中断 更新
- 60秒ポーリング + 手動リフレッシュ
- エラー通知 / ログ

## 追加候補
- フィルタ条件UI（進行中定義の変更）
- 自動起動（Login Item）
- 通知/ショートカット

## 実装順序（マイルストーン）
1. 設定/Keychain/Notionクライアント基盤
2. トレイ + UI（表示/更新）
3. 監視・エラーハンドリング強化
4. 追加候補の順次実装
