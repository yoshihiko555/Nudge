# 概要

## 目的
Notion の「進行中タスク」を macOS メニューバー常駐アプリで即時確認・更新できるようにする。

## ゴール
- メニューバーから小窓を開き、進行中タスク一覧を表示する
- タスクを「完了」または「中断」に更新できる
- ローカル完結（サーバ/アプリ内DBなし）

## 非ゴール
- チーム共有・複数ユーザー対応
- オフラインでのタスク編集
- Notion 以外の連携

## 対象ユーザー / ユースケース
- 個人のタスク管理者
- すぐに「今やること」を見たい、完了を即反映したい人

## 主要機能
- Notion 連携設定（トークン/Database ID/Status値 ※更新用・UI非表示）
- 進行中タスク取得・表示
- 完了/中断の更新
- 手動リフレッシュ / 自動更新
- 失敗時の通知と再試行

## 前提 / 仮定 / 未決事項
### 前提
- Notion のタスクは Database で管理されている
- Notion API は Data Source を前提とする最新仕様を使用する（API Version を固定）
- Database は Integration と共有済み

### 仮定
- 進行中/完了/中断の状態値は Notion 側に存在する
- 画面上に Status を表示しない（デフォルトは「進行中のみ」表示）

### 決定事項
- 中断タスクの「戻す」はメニューから「中断一覧」へ一時切替して実行する
